NOTE: When array is rotated by multiples of N, it becomes original array

So, d = d%n

brute:

  • Copy d elements from array to temp
  • Move elements from d to n-1 to 0 - d-1
  • Copy back d elements to end of the array
  • Time complexity: o(n+k)
  • Space complexity: o(k)

  • TC: O(N)

  • Search for element in arr2
  • Have visited array for arr2
  • If elements are same and element is arr2 is not taken
  • Then add the element to ans and mark element in arr2 as taken
  • When element in arr2 becomes greater than element1, then we can stop. Since array is sorted. Element1 will definitely not have pair.
  • Time: o(n1 x n2)
  • Space: o(n2 + (n1+n2)) [ visited arry of size n2 ]

  # a = 1 2 2 3 4

  # b = 2 2 3 3 5 6 7

  # v = 0 0 0 0 0 0 0

  visited = [0] * m

  ans = []

  for i in range(n):

      num1 = arr[i]

      for j in range(m):

          num2 = brr[j]

              break

              break

  return ans

def longestSubarrayWithSumK(a: [int], k: int) -> int:

 

   n = len(a)

   max_length = 0

   for i in range(n):

       for j in range(i, n):

           sum = 0

           for l in range(i,j+1):

               sum += a[l]

           if sum == k:

               max_length = max(max_length, j-i+1)

   return max_length

def longestSubarrayWithSumK(a: [int], k: int) -> int:

 

   # 1 2 3 1 1 1

   # 2 0 0 0 3

   n = len(a)

   index = {}

   prefix_sum = 0

   max_length = 0

   for i in range(n):

       prefix_sum += a[i]

       if prefix_sum == k:

           max_length = i+1

     

       diff = prefix_sum - k

       if diff in index:

           length = i - index[diff]

           max_length = max(max_length, length)

     

       if prefix_sum not in index:

           index[prefix_sum] = i

     

   return max_length

Optimal: 2 pointer approach for positive and zeros

  • Have 2 pointers left and right
  • If the sum <= k keep adding right elements
  • If the sum > k then keep removing left elements
  • If sum == k then update maxLen
  • Time: o(2n)
  • Space: o(1)
  • Works only for array with positives and zeros

def longestSubarrayWithSumK(nums: [int], k: int) -> int:

 

   # 1 2 3 1 1 1 1 6

   n = len(nums)

   left = 1

   right = 1

   sum = nums[left]

   maxLen = 0

   # 1 2 1 3

   while right < n:

       while(left<=right and sum > k):

           sum -= nums[left]

           left += 1

       right += 1

       if(right < n):

           sum += nums[right]

 

   return maxLen
